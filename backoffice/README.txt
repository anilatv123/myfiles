Introduction
------------

This block is mainly created to act as a back office block. At present it can be used to add the threshold value for a particular course.

Threshold Value
---------------

The threshold value contains the following.

Post_high: Maximum number of posts.
Post_low:  Minimum number of posts.
Login_high:Maximum number of Logins.
Login-low: Minimum number of Logins.
Aprov_high:Maximum grade for that course.
Aprov_low: Minimum grade for that course.

Features
--------

This block is intended to be used only by teachers, so students aren't going to see it.

Teachers can use this tool to store a threshold value for a course. This block allows to update the threshold value also.
